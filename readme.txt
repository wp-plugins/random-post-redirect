== Installation ==

1. Upload the `random-post-plugin` folder to the `/wp-content/plugins/` directory
2. Active the Random Post plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `Random Post` menu that appears in your admin menu
